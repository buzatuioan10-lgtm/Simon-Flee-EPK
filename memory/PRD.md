# Simon Flee DJ Presskit - PRD

## Original Problem Statement
Create a presskit website for DJ project "Simon Flee" based on Google Docs information (bio, layout) and provided artist photo.

## User Persona
- **Primary:** Club promoters, booking agents, festival organizers
- **Secondary:** Music industry professionals, media/press

## Core Requirements (Implemented)
- [x] Hero section with artist photo banner
- [x] Bio section with highlighted genre tags
- [x] Embedded Mixcloud player (Showcase Mix)
- [x] Embedded SoundCloud player (Unreleased Playlist)
- [x] Social links (Bandcamp, Mixcloud, SoundCloud, YouTube, Instagram, Linktree)
- [x] Tech Rider table (equipment requirements)
- [x] Contact section (Phone, WhatsApp, Email, Instagram)
- [x] Footer with copyright
- [x] Responsive design (mobile-friendly)
- [x] Genre marquee animation
- [x] Minimal bright design (Swiss Punk / Light Brutalist aesthetic)

## Implementation Date
- **Initial Build:** February 2026

## Tech Stack
- React (frontend)
- Framer Motion (animations)
- TailwindCSS (styling)
- No backend required (static site)

## Design System
- **Colors:** Off-white (#F2F2F2), Black (#0A0A0A), Acid Green (#D1FF00)
- **Fonts:** Space Grotesk (headings), JetBrains Mono (body/data)
- **Style:** Sharp edges, brutalist grid, high contrast
